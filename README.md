# Desafio Target

## Challenge 1 - Class Sales
<br>

**calculateComissions()** --> verifica todas as vendas registradas no arquivo json e retorna a comissao proporcional às vendas de cada funcionário.<br><br>

## Challenge 2 - Class Register
<br>

**registerItem(Código, Nome)** --> Recebe o código do produto e o nome do produto e registra no arquivo json com todos os itens do estoque.<br>

**addItem(Código, Quantidade)** --> Recebe o código do produto e a quantidade a ser adicionada no estoque.<br>

**removeItem(Código, Quantidade)** --> Recebe o código do produto e a quantidade a ser removida do estoque.<br>

**getLogs()** --> Retorna a lista de todas as modificações feitas no estoque.<br><br>

## Challenge 3 - Class Calculator
<br>

**calculateTaxes(Valor, Vencimento)** --> Recebe o valor da cobrança e a data de vencimento e retorna os juros baseados na quantidade de dias passados ao vencimento.
